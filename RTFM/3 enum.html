Перечисляемый тип

Перечисляемый тип (англ. enumerated type, enumeration, enum) — тип данных, состоящий из множества именованых значений, называемых элементами или членами.

Использование в языке Си

enum WeekDay { Monday, Tuesday, Wednesday, Thursday, Friday, Saturday, Sunday }; int isWeekend(enum WeekDay day) { return day == Saturday || day == Sunday; }

Представление в памяти

В памяти значение перечисляемого типа хранится в виде числа, соответствующего нужному элементу. Данное число, в зависимости от языка программирования, может быть сгенерировано автоматически, либо задано вручную.

enum TogglePosition { On = 1, Off = 0 };

Преобразование enum в строку

Поскольку в памяти хранится лишь числовые идентификаторы элементов, непосредственное получение строки «On» по значению 1 в языке Си невозможно. Однако, при желании, можно написать код, получающий строковые эквиваленты значений перечисляемого типа. Гугл: «c enum to string».
